# Rishabh Provision Store

A premium MERN stack Grocery Store Application featuring:
- Role-based Authentication (Admin/Customer)
- Fully functional Wallet, Coupons, Reward Points, Notifications, Invoices, and Account Settings
- Premium responsive glassmorphic UI built with Next.js, Tailwind CSS, Framer Motion, and Lucide Icons
- Seeding scripts for demo users and inventory

## Structure
- `/frontend`: Next.js 14 Web Application
- `/backend`: Node.js Express API server
- `/admin`: Admin interface
